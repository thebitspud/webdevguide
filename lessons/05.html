Lesson 5: Styling and Attributes


So far, we have learned about how to create HTML pages from scratch and add various types of content using both inline and block-level elements. In this lesson, we will be learning about utilizing HTML divisions, attributes, and values. Along with that, we will begin working with CSS stylization to alter the presentation of content on a website.

5.1: Styling Attributes

In our last few lessons, we worked with some basic attributes in HTML. Attributes are used in tags to provide contextual information to elements and follow the name/content convention pattern of attribute="value". A tag may have more than one assigned attribute and in certain cases (such as with class="...") an attribute may extend multiple values. Some of the attributes we have already covered in this course include href="...", src="...", width="...", and height="...".

One very useful attribute we have not yet utilized is the style="..." attribute. Styles use inline CSS (which we will learn more about later on) to change the appearance of specific elements. Multiple content values can be used with a style attribute by separating them with a ;

...
	<body>
		...
		<p style="background-color:lightblue">
			The background of this paragraph is blue now!
		</p>

		<p style="background-color:black; color:white">
			This paragraph has white on black text now!
		</p>
		...
	</body>
...

Above: Using the style="..." attribute.

Another attribute we have not covered so far is the class="..." attribute, which specifies styling for multiple similar elements. To declare the style of a class, you must use either an internal style sheet and the <style> tag or an external style sheet in a separate .css file that is referenced by a <link> tag. We will be covering the specifics of style sheets in our next lesson (copy/pasting CSS elements will work for now). Regardless of which you choose, you need to have a CSS selector that contains .classname { within your style sheet.

...
	<head>
		...
		<style>
			.bg-red {
				color: white; 
				background-color: red;
				font-family: Verdana, Tahoma, sans-serif;
				padding: 10px;
			}
		</style>
		...
	</head>

	<body>
		...
		<p class="bg-red">
			This paragraph is red with white text!
		</p>
		...
	</body>
...

Above: Using the class="..." attribute and an inline style sheet to change how an HTML file is displayed.

Similarly, the id="..." attribute can be used with either an internal or external style sheet for specifying the content of a single element. HTML IDs typically used to identify unique sections on a page to be used with JavaScript. However, IDs will not work properly if you specify the same ID for multiple elements. The CSS selector for an ID is #id-name {

...
	<head>
		...
		<style>
			#bg-gray {
				color: white; 
				background-color: gray;
				font-family: Verdana, Tahoma, sans-serif;
				padding: 2%;
			}
		</style>
		...
	</head>

	<body>
		...
		<p id="bg-gray">
			This paragraph is gray with white text!
		</p>
		...
	</body>
...

Above: Using the id="..." attribute and an inline style sheet to change how an HTML file is displayed.

5.2: Spans and Divisions

<span>...</span> : Used for specifying an inline section within a document that has modified CSS. The <span> tag effectively acts as a blank slate that does not have any predesignated attributes of its own. 

...
	<style>
		.text-green {
			color: green;
			font-family: monospace;
			font-size: 13px;
		}
	</style>
...
	<p>
		Look at this <span class="text-green">modified text</span>!
	</p>
...

Above: Using the <span> tag to change the color of a specific text segment.

<div>...</div> : Divisions are empty block-level containers that are used to alter entire sections of a web page. A <div> element is the block-level equivalent of a <span> tag. 

...
	<style>
		.bg-black {
			color: white;
			background-color: black;
			font-family: monospace;
			padding: 10px;
			margin-top: 100px;
			margin: auto;
		}
	</style>
...
	<div class="bg-black">
		Divisions are very cool and can be used in many interesting ways!
	</div>
...

Above: Using the <div> tag to modify an HTML block. More complex use cases will be covered in future lessons.

5.3: Miscellaneous HTML

<iframe src="...">...</iframe> : An inline frame that is used to embed and display content directly from another website or document. It is recommended to specify a size for the <iframe> element either directly in the opening tag or in a style sheet elsewhere. If you insert text between the opening and closing tags or add a name="..." attribute, it will be displayed if the <iframe> is unable to properly show the website.

...
	<body>
		<iframe src="https://vikingsdev.github.io/" width="800" height="500">VikingsDev</iframe>
	</body>
...

Above: Using the <iframe> element to embed a website.

The following elements are all used to specify <head> element data.

<meta> : Used to specify metadata about the page. Content within a <meta> tag serves various special functions such as specifying a document charset or helping search engines find and index pages.
<style>...</style> : Defines an internal CSS style sheet.
<script>...</script> : Links to an external JavaScript file
<link href="..." rel="..." type="..."> : Links to an external file (CSS files, icons, CDN libraries, etc.)

...
	<head>
		<meta charset="utf-8">
		<meta name="author" content="Thebitspud">

		<style>
			body {
				color: white;
				background-color: blue;
				font-family: monospace;
			}
		</style>

		<script>
			document.getElementById("center-div").innerHTML="Is this JavaScript?";
		</script>

		<script src="script.js" type="text/javascript"></script>

		<link href="style.css" rel="stylesheet" type="text/css">
	</head>
...

Using the <meta>, <style>, <script>, and <link> elements.
